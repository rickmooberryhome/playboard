# PlayBoard Homepage Starter

This is an original PlayBoard homepage concept inspired by the premium, dark, performance-product feel of WHOOP.

## Files

- `index.html` — Static homepage
- `styles.css` — Full responsive styling
- `script.js` — Mobile nav and starter form behavior
- `README.md` — Notes and next steps

## Design direction

- Dark performance-style background
- Large uppercase hero typography
- Bright CTA accent
- Dashboard-style recruiting metrics
- Parent trust sections
- Athlete task view
- Blog/article card system
- Intake form starter

## Important

This does not copy WHOOP assets, images, logos, or proprietary content.

It uses the general design direction: premium, dark, data-driven, athlete-focused.
